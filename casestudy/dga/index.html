The Challenge

Architects Billing $200/hr Doing Data Entry

Diehl Group Architects specializes in forensic architecture — investigating building failures, construction defects, and code compliance issues. Their work generates a massive volume of audio that needs transcribing: interviews, depositions, site visit recordings, and client calls.

On an average week, the team processes around 200 audio files. On larger contracts, that number climbs past 1,000. Every one of those files was being transcribed manually — by architects billing at $200 an hour.

A single transcription batch could eat an entire work day. Sometimes longer. That meant senior staff spending their time typing instead of doing the forensic analysis they were hired for. The firm was paying top dollar for low-value repetitive work, and the backlog only grew as contracts stacked up.

The Solution

A Local AI Transcription Tool Built for the Job

Little Hammer Labs built DGA a custom transcription tool powered by OpenAI's Whisper model, designed to run entirely on the firm's local hardware. No audio files leave the building — a requirement for a firm handling sensitive legal and forensic materials.

The tool processes WAV files in batch, handling hundreds or thousands of recordings in a single run. It includes a professional desktop interface for day-to-day use and a headless mode for automated processing. Built-in features like copy detection and custom dictionary processing ensure the output matches the firm's terminology and standards. Transcriptions export to both TXT and CSV formats, slotting directly into DGA's existing documentation workflow.

A working version was delivered in two weeks. Over the following month, Little Hammer Labs refined the tool through collaborative sessions with DGA's team — adjusting the interface, output formatting, and workflow integration until it fit their process exactly. Six weeks from kickoff to a finished product the team actually wanted to use.
